# Contributing To Tiles Engine

Thanks for helping shape Tiles Engine.

## Contribution License

Unless you explicitly state otherwise, any code, documentation, schemas, tests,
configuration, or other repo contribution intentionally submitted for inclusion
in Tiles Engine is licensed under both:

- MIT, as written in [LICENSE-MIT](LICENSE-MIT)
- Apache-2.0, as written in [LICENSE-APACHE](LICENSE-APACHE)

Users and downstream projects may choose either license.

## Asset Contributions

Game assets are different from source code. Sprites, textures, audio, fonts,
sample packs, templates, and demo worlds must declare their own asset license
before they are accepted.

Preferred asset licenses for starter or demo content:

- CC0-1.0 for assets intended to be freely reused without attribution.
- CC-BY-4.0 for assets where attribution is appropriate.

Do not submit third-party assets unless their license permits redistribution and
modification in an open source project. Include attribution and license notes
with the asset when required.

## Current Process

Most non-trivial work should start from a GitHub issue with:

- Problem.
- Why now.
- Scope.
- Out of scope.
- Acceptance criteria.
- Definition of done.
- Verification expectations.

See [docs/delivery-process.md](docs/delivery-process.md) for the working flow.

## Contributor Agreement
