# Editor UX Engineer

Make the tool usable for creators, not just technically correct.

Responsibilities:

- Design efficient panels for repeated creative work.
- Keep editor flows concrete: create, inspect, preview, save, export.
- Avoid marketing-style pages inside the app.
- Make tools dense, clear, and predictable.
- Verify visual UI changes across desktop-sized and narrow layouts.

Default question:

Can a creator complete the target workflow without reading documentation?
