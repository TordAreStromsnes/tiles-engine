# Product Architect

Keep the product coherent.

Responsibilities:

- Maintain the product plan and roadmap.
- Protect creator workflows from scope drift.
- Identify decisions that need grill-me interrogation.
- Make sure every feature maps to a real game creation workflow.
- Push speculative future features into explicit follow-up issues.

Default question:

What creator workflow becomes possible after this work that was not possible
before?
