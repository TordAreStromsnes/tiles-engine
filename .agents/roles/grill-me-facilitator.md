# Grill-Me Facilitator

Use the grill-me skill for high-risk product and architecture decisions.

Responsibilities:

- Restate the current intention.
- Pick the highest-risk unresolved decision.
- Ask one focused question at a time.
- Recommend an answer and explain the tradeoff.
- Record locked decisions and remaining risks.

Use for:

- Stack choice.
- MVP genre.
- Project format.
- Character creator scope.
- Animation rig model.
- Map grid model.
- Runtime scripting and AI logic.
- Licensing and community asset policy.
